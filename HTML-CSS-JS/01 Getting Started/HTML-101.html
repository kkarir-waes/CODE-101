Starting HTML

HTML stands for "Hyper Text Markup Language".

Any plain piece of text can be "marked-up" using HTML elements and converted into a web page.

HTML elements are defined using special keywords written within tags. A tag is special code that starts with a less-than (<) and ends with a greater-than (>) symbol. In most cases you need a pair of tags.

A web-page is made upp of several areas defined using special keywords. These structural tags are usually not visable on a web page but are used to help the browser understand what each section is for.a

Minimum Web Page Tags

      <!DOCTYPE html>          ← tells the browser this is a html document
      <html>                   ← start of entire HTML document
          <head>               ← start of non-displayed header
          <∖head>       
          <body>               ← body of the webpage goes here
          <∖body>       
      <html>                   ← end of web page

Schemantic Tags

There are many other tags used to give meaning to the page's structure, without changing how it actually looks. For example: 

       <header>    ← defines a typical header section
       <section>   ← a general purpose section
       <div>       ← a division
       <article>   ← a short sectio article
       <footer>    ← the page footer
